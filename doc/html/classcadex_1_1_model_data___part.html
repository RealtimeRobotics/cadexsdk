Detailed Description

Defines a leaf node in the scene graph hiearchy.

A part corresponds to mechanical part or a product in the product structure hieararchy.

A part contains one or several representations: zero or one B-Rep representation, and/or zero, one or multiple polygonal representations.

See also
Product Structure, Part representations.
Examples:
advgeom/meshsimplify/main.cxx, exploring/appearance/main.cxx, exploring/bom/main.cxx, exploring/breprepresentation/main.cxx, exploring/layers/main.cxx, exploring/pmi/main.cxx, exploring/polyrepresentation/main.cxx, exploring/propertytable/main.cxx, meshing/lods/main.cxx, meshing/remeshing/main.cxx, modeling/assembly/main.cxx, modeling/brep/main.cxx, modeling/brepfeatures/main.cxx, modeling/brepprimitives/main.cxx, modeling/metadata/main.cxx, modeling/poly/main.cxx, visualization/qtquick_qml/pmiviewer/main.cxx, and visualization/qtquick_qml/selectionhandling/main.cxx.
