Detailed Description

Defines an RGBA color (with alpha channel).

Components must be within a [0, 1] range.

Examples:
exploring/polyrepresentation/main.cxx, modeling/assembly/main.cxx, modeling/metadata/main.cxx, modeling/poly/main.cxx, visualization/offscreen/main.cxx, visualization/qtquick_qml/measurements/main.cxx, and visualization/qtquick_qml/pmiviewer/main.cxx.
