Detailed Description

Defines parameters of the STEP_Reader.

Refer to Importer Parameters.

Parameters can be set using STEP_Reader::Parameters().

Examples:
conversion/import/main.cxx, exploring/pmi/main.cxx, visualization/offscreen/main.cxx, and visualization/qtquick_qml/baseviewer/main.cxx.
