Detailed Description

Provides information about external files not found during import.

Implements iterator interface. Each element returned by this iterator is an instance of Base_MissingExternalReference class, which provides access to the name of external file not found and its parent scene graph element.

Currently the functionality of finding missing external references is implemented only for STEP, JT and Solidworks formats, as they are the only formats in CAD Exchanger that support external references.

The information about missing external refrences can be retrieved after Base_Reader::Transfer has completed.

Note
For JT this information can be retrieved right after Base_Reader::Transfer completion only if JT_ReaderParameters::DelayedConversion is false (it is false by default).
This functionality is currently in the preview state. Source and binary compatibility may be broken. To use it, please define __CADEX_PREVIEW_READ_STATISTICS macro to be 1.
