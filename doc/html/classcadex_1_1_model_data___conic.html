Detailed Description

Base class for 3D conic curves.

A conic curve has an axis placement returned by Position(). The curve lies on a plane perpendicular to the Z-axis of the axis placement. Parametrization is defined taking into account X-axis of the axis placement.

See also
ModelData_Axis2Placement, ModelData_Conic2d.
