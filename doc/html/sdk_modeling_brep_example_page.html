Demonstrates creation of all curve and surfaces types, all topological entities (vertex, edge, face, etc) and all body types (solids, sheet, wireframe).

Description

In this example we are using every available geometric primitive to create edges, faces and bodies.

Edges are represented by line, circle, ellipse, parabola, hyperbola, offset curve, bezier curve and b-spline curve.

Faces are represented by plane, sphere, cylinder, cone, torus, surface of linear extrusion, surface of revolution, offset surface, bezier surface and b-spline surface.

Bodies are represented by bodies types: solid, sheet, wireframe and acorn.
