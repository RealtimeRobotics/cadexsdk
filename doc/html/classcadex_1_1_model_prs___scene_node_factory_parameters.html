Detailed Description

Defines parameters which using by ModelPrs_SceneNodeFactory.

A factory parameters class contains a shared pointer to its internal implementation, so simple copying an object will just copy a shared pointer. Therefore any modification of the copied object will propagate to original object.

bool cadex::ModelPrs_SceneNodeFactoryParameters::Decomposition ( ) const

Returns a decomposition.

When this flag is true then each ModelData_Representation will exploded on ModelData_Body (ModelData_BRepRepresentation) or ModelData_PolyVertexSet (ModelData_PolyRepresentation) and from each will created a scene node.

This is may equivalent to this code:

ModelData_BRepRepresentation aBRep = ...;
ModelPrs_SceneNode aBRepNode;
const auto& aList = aBRep.Get();
for (std::size_t i = 0; i < aList.Size(); i++) {
auto aBodyNode = myFactory.Create (aList[i], aBRep);
aBRepNode.AddChildNode (aBodyNode);
}

If decomposition is disabled (this flag is false) then for each ModelData_Representation will created only one scene node.
