Detailed Description

Defines a background style for usage in qml code.

The following code snippet demonstrates an usage:

ModelPrsQtQuick_ViewPort {
backgroundStyle: ModelPrsQtQuick_BackgroundStyle {
topColor: "#303030"
bottomColor: "#303030"
}
}
