Detailed Description

Defines a Unicode (UTF-16) string wrapping a standard string.

Examples:
exploring/bom/main.cxx, exploring/pmi/main.cxx, exploring/propertytable/main.cxx, exploring/transformations/main.cxx, meshing/remeshing/main.cxx, misc/progressindicatorgui/main.cxx, modeling/brep/main.cxx, modeling/elementremoval/main.cxx, and visualization/qtquick_qml/selectionhandling/main.cxx.
